# 02 자료형

## 02-1 숫자형

### 정수형
양의 정수, 음의 정수, 숫자 0..

### 실수형
실수형, 소수점이 포함된 숫자

지수표현방식 
```python
a = 4.24E10
```
4.24E10 = 4.24 * 10<sup>10</sup>
4.24e-10 = 4.24 * 10<sup>-10</sup>

### 8진수/16진수
8진수를 만들기 위해 0o또는 0O를 붙여준다.

16진수는 0x로 시작하면 된다.

### 숫자형을 활용하기 위한 연산자

- 더하기 \+ 
- 빼기 \- 
- 곱하기 \* 
- 나누기 \/ 
- 제곱연산 \*\* 
- 나누기 후 나머지 \%
- 나누기 후 몫 \/\/
